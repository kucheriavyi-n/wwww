Мы

"C Technology" является единственной местной компанией, которая обеспечивает разработку программного обеспечения для банковских институтов.

В прошлом, Демирбанк получал услуги по разработке программного обеспечения посредством аутсорсинга.

Два года назад, Демирбанк принял решение создать локальную команду разработчиков программного обеспечения, в городе Бишкек, для разработки программного обеспечения в частности для своих нужд.

В результате, были наняты сотрудники, включая руководителя программных разработок для Демирбанка. В течении этого времени, разработчики приняли участие в нескольких тренингах, на протяжении 1,5 года командой было завершено несколько проектов включая приложение «Мобильный Банкинг».

Изначально, основной задачей для команды разработчиков было усовершенствование программного обеспечения для Демирбанка, задачу группа выполнила успешно, затем Демирбанк принял решение, использовать потенциал команды для изучения возможных потребностей других банков находящихся в группе компаний и местных банков Кыргызстана. Но согласно законодательству Демирбанк не может оказывать услуги кроме финансовых. С этой целью было решено создать независимую компанию, а также дополнительно нанять новых сотрудников. В данный момент компания «C Technology» состоит в Парке Высоких Технологий и приносит на рынок современные и успешные продукты в сфере высоких технологии. Компания была создана в 2015 году.
